# Email Scheduler

Automated email notification system using GitHub Actions.

## Projects
- 📊 FD Maturity Alert
- 💰 Expense Report
- 🎂 Birthday Reminder

## Setup
1. Clone the repo
2. Add secrets in GitHub Settings
3. Push to main branch
4. Workflow runs automatically

## Secrets Required
- EMAIL_SENDER
- EMAIL_RECEIVER
- PASSWORD_API_KEY
